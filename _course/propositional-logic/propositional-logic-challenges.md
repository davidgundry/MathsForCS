## Truth Tables

Write a function that can be used to generate a full truth table for each of the following expressions both with and without using logical connectives (!, &&, \|\|):

* ¬A
* A∧B
* AvB
* A⊕B
* A⇒B
* A⇔B
* (A∧B)vC
